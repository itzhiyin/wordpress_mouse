# 插件概述
使用此插件可一键自定义Wordpress前端鼠标指针样式。利用该插件，站长可以快速实现替换多种鼠标指针样式于网站前端。
鼠标图案均来自于网络，插件仅作收集整理，插件完全开源无任何商业性质。
