== Description ==

Wordpres自定义鼠标样式插件 由<a href="https://itzhiyin.com>宁子</a>开发，一键自定义Wordpress前端鼠标指针样式。利用该插件，站长可以快速实现替换多种鼠标指针样式于网站前端。


== Installation ==

1. 上传 `wordpress_mouse`目录 到 `/wp-content/plugins/` 目录
2. 激活插件，开始设置使用。
